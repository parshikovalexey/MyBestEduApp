## Описание продукта

AI-платформа для формирования навыка изучения материала у детей, превращающая обучение в ролевую игру с уровнями, боссами, ачивками и инвентарем.

**Ключевое отличие:** Фокус не на передаче знаний, а на формировании НАВЫКА ОБУЧЕНИЯ.

---

## Геймификация

### Игровые элементы

| Элемент учебника | Игровой аналог |
|-----------------|---------------|
| Глава/параграф | Уровень |
| Осваиваемая тема | Босс |
| Проверочная работа | Битва с боссами |
| Справочные материалы | Инвентарь персонажа |
| Сформированные навыки | Ачивки |

### Как это работает

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   ИГРОВОЙ ОПЫТ                                            │
│                                                             │
│   Ребенок играет в ролевую игру, где:                     │
│   • Герой изучает мир (читает материал)                    │
│   • Попадает на боссов (проверочные работы)               │
│   • Собирает инвентарь (артефакты, формулы, даты)         │
│   • Получает ачивки (навыки и знания)                     │
│   • Прокачивается (становится умнее)                       │
│                                                             │
│   НО: Каждая игровая механика = образовательный процесс    │
│                                                             │
│   Пример:                                                  │
│   • "Собери автомобиль для перевозки груза"               │
│   • Для этого рассчитай мощность двигателя                  │
│   • Пополняешь инвентарь формулами                        │
│   • Применяешь знания на практике                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### Ачивки и прогресс

- **Навык изучения темы** - "Исследователь"
- **Навык создания артефактов** - "Архивариус"
- **Понимание** - "Мастер объяснений"
- **Практика** - "Боец"
- **Рефлексия** - "Философ"

## Целевая аудитория

### Primary (дети)
- **Возраст:** 10-16 лет
- **Интересы:** Игры, ролевые приключения
- **Боль:** Скучное образование, отсутствие мотивации

### Secondary (родители)
- **Интересы:** Развитие ребенка, видимость прогресса
- **Боль:** Не понимают что происходит, не знают как помочь
- **Готовность платить:** Да, если увидят ценность

## Технические требования

### Core Features MVP

1. **Цикл изучения**
   - Работа с материалом (текст, изображения)
   - Выделение основного и непонятного
   - AI-пояснения через диалог
   - Проверочные работы
   - Рефлексия

2. **Артефакты**
   - Автогенерация конспекта
   - Справочные материалы
   - Резюме от тьютора
   - История сложностей

3. **Кривая забывания**
   - Персональный spaced repetition
   - Напоминания о практике
   - Прогресс по retention

4. **Базовая геймификация**
   - Уровни и темы
   - Инвентарь
   - Ачивки
   - Прогресс-бар
