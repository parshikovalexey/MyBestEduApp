## Описание продукта

Масштабная ролевая игра с регулярными эпизодами, онлайн-сообществом, стримами и трансляциями. Персонализированный образовательный контент, который дети ждут как сериал.

**Ключевое отличие:** Не просто инструмент обучения, а полноценное развлекательное образовательное шоу.

## Ключевые особенности

### 1. Персонализация

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   КАЖДЫЙ ИГРОК ПОЛУЧАЕТ:                                  │
│                                                             │
│   • Персональные квесты под свой уровень                   │
│   • Адаптивные задания                                     │
│   • Кастомизированный герой                                │
│   • Собственная история прогресса                          │
│                                                             │
│   НО: В рамках общего сюжета и мира                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2. Онлайн-сообщество

- **Guilds/Кланы** - группы по интересам
- **Совместные квесты** - collaboration
- **Leaderboards** - но осторожно (мотивация vs соревнование)
- **Стримы** - создатели контента играют с детьми
- **Fan art** - творчество на основе игры

### 3. Образовательный сторителлинг

```
Пример сюжета:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Герой должен построить город для беженцев.

Чтобы построить дома - нужна математика (площадь, периметр)
Чтобы провести воду - нужна физика (давление, трубы)
Чтобы выбрать место - нужна география (климат, ресурсы)
Чтобы договориться - нужны soft skills

И все это встроено в геймплей, а не как "урок"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## Риски и реалистичность

### Почему это сложно

| Фактор | Сложность | Пояснение |
|--------|----------|-----------|
| Production cost | 🔴🔴🔴🔴🔴 | AAA-quality стоит миллионы |
| Content velocity | 🔴🔴🔴🔴 | Нужен постоянный поток контента |
| Community building | 🔴🔴🔴 | Нужна критическая масса |
| Competition | 🔴🔴🔴🔴 | Roblox, Minecraft огромны |
| Retention | 🔴🔴🔴 | F2P игры живут ~30 дней |

### Почему это возможно

| Фактор | Шанс | Пояснение |
|--------|------|-----------|
| Education niche | 🟢 | Ниша未被垄断 (незаполненная) |
| AI-powered content | 🟢 | AI может ускорить production |
| Edutainment trend | 🟢 | Растет после COVID |
| Passion project | 🟢 | Команда верит в продукт |
