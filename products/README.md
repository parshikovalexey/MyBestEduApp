## Product 1: Study Skill Trainer ⭐ C2C VISION

**Описание:** AI-платформа для формирования навыка изучения материала, превращающая обучение в ролевую игру.

### Ключевые элементы:
- Цикл изучения темы (ознакомление → обработка → проверка → рефлексия)
- AI-тьютор для персонализированной помощи
- Spaced repetition через кривую забывания
- Геймификация (уровни, боссы, ачивки, инвентарь)
- Parent Dashboard для вовлечения родителей

### Бизнес-модель:
- Freemium (базовый доступ)
- Parent Premium ($5-15/month)
- Реклама (minimal)

### Timeline: Year 2+ (после B2B валидации)

## Product 3: B2B Learning Solutions 💰 PRIMARY FOCUS

**Описание:** Заказные образовательные решения для бизнеса с фокусом на формирование НАВЫКОВ, а не просто передачу знаний.

### Ключевые элементы:
- Corporate Training Games (с retrieval practice, spaced repetition)
- Onboarding Programs
- Compliance & Safety Training
- Leadership Development

### Бизнес-модель:
- Fixed price projects ($30K-500K)
- Retainers ($15K-50K/month)
- Enterprise contracts

### Timeline: NOW (Year 1)

### Почему сейчас:
- Revenue для финансирования C2C development
- Transferrable expertise
- Real-world validation
- Faster feedback loop чем C2C

### Педагогические основы:
- Retrieval Practice (практика извлечения)
- Spaced Repetition (распределенное повторение)
- Interleaving (чередование)
- Elaboration (углубление)
- Формирование артефактов

## Decision Points

### Why B2B first?

**We chose B2B Learning Solutions as PRIMARY because:**
- Faster revenue (vs C2C which takes longer to monetize)
- Higher deal sizes ($30K-500K vs $5-15/month)
- B2B sales cycle is shorter than C2C product-market fit
- Real-world validation of pedagogical approach
- Revenue funds C2C development
- Can leverage existing network

**But:** B2B has its own challenges
- Enterprise sales cycle (3-9 months)
- Need business development skills
- Custom delivery is resource-intensive

### When to start C2C (Study Skill Trainer)?

**Recommendation:** Year 2
- After B2B validates methodology
- After first revenue
- When ready to invest in product development

**Alternative:** Start in parallel if:
- Have existing network for B2B
- Have budget for C2C development
- Want to test both tracks

### When to build Content Engine?

**Recommendation:** Year 2
- After understanding content needs from B2B
- Need automation to scale
- Can build faster with real content examples

### When to pursue Epic Game?

**Recommendation:** Year 3+ only
- Requires proven pedagogy
- Requires significant funding
- Requires full production team
- High risk without validation
