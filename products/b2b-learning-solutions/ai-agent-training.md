# Project: AI Agent Engineering & VibeCoding Training

**Дата создания:** 09.04.2026
**Статус:** PRIMARY B2B PROJECT
**Client:** Software Development Company

---

## Overview

Обучение сотрудников IT-компании использованию AI агентов и VibeCoding для повышения продуктивности и качества работы.

**Целевая аудитория:** 2 группы
1. **Проект-менеджеры** - создание мини-продуктов без команды разработки
2. **Разработчики** - интеграция AI в процессы с гарантией качества

### AI-куратор и тьютор (встроен в платформу)

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   AI-ТЬЮТОР ВНУТРИ ПЛАТФОРМЫ:                         │
│                                                             │
│   🤖 AI-куратор:                                         │
```

```
│      • Мониторинг активности AI                          │

│   4. МОДЕЛИ - ПЛАТНЫЕ И БЕСПЛАТНЫЕ                     │
```
