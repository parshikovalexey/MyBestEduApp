### Gameplay Overview

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   GAME: "SAFE SITE"                                       │
│                                                             │
│   ЖАНР: 3D Simulation / Decision Game                    │
│   PLATFORM: Mobile + Web                                  │
│   TIME TO COMPLETE: 30-45 minutes (base course)         │
│                                                             │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   ГЕЙМПЛЕЙ:                                        │  │
│   │                                                     │  │
│   │   Игрок = new сотрудник на стройплощадке          │  │
│   │                                                     │  │
│   │   КАЖДЫЙ УРОВЕНЬ:                                  │  │
│   │   1. Ситуация (видео/3D сцена)                     │  │
│   │   2. Вопрос "Что не так?"                          │  │
│   │   3. Обучающий контент                             │  │
│   │   4. Симуляция решения                             │  │
│   │   5. Результат + объяснение                         │  │
│   │   6. Рефлексия                                      │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### Game Elements

| Element | Description |
|---------|-------------|
| **Levels** | Ситуации безопасности (10 основных) |
| **Boss Fights** | "Critical incidents" - complex scenarios |
| **Inventory** | Safety checklist, equipment guide |
| **Achievements** | Safety certifications |
| **XP System** | Points for correct decisions |
| **Leaderboard** | Team-based (not individual!) |
| **Daily Missions** | Spaced repetition challenges |

### Level Structure

```
LEVEL 1: "FIRST DAY"
━━━━━━━━━━━━━━━━━━━━
Context: Employee arrives at construction site
Challenge: Identify all safety equipment needed
Learning: PPE requirements
Consequence: If missed items → tutorial about why each is needed

LEVEL 2: "WORKING AT HEIGHTS"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Context: Task requires ladder work
Challenge: Proper ladder setup and safety
Learning: Fall protection requirements
Consequence: Wrong choice → "You fell!" scenario (not graphic)

LEVEL 3: "ELECTRICAL HAZARDS"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Context: Working near power lines
Challenge: Identify all electrical dangers
Learning: Electrical safety protocols
Consequence: Mistakes → "Electrical incident" simulation

... and so on for 10 levels

BOSS LEVEL: "MAJOR INCIDENT"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Context: Multiple hazards simultaneously
Challenge: Prioritize and respond correctly
Learning: Emergency response
Consequence: Realistic emergency scenario
```
