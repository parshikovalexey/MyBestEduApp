## МАТЕРИАЛ 1: Техника "5 единичек"

### Суть методики

Переход из Фазы 1 (ресурсы = крохи) в Фазу 2 (система даёт ресурсы) возможен только через **фокус**.

### Ключевые принципы

1. **Одна целевая аудитория** — выбрать только одну ЦА
2. **Обмен ресурсов** — крохи, копейки, энергия → система даёт много ресурсов
3. **Фокус** — главный двигатель перехода между фазами

### Чек-лист для применения техники

```

│  ТЕХНИКА "5 ЕДИНИЧЕК" — ЧЕК-ЛИСТ                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ☐ 1. ЦЕЛЕВАЯ АУДИТОРИЯ (ТОЛЬКО ОДНА)                      │
│     ☐ Определена ли одна конкретная ЦА?                        │
│     ☐ Не смешаны ли разные ЦА в одну?                          │
│     ☐ Если несколько — выбрана приоритетная?                    │
│                                                                 │
│  ☐ 2. ФОКУС                                                   │
│     ☐ Все ресурсы сконцентрированы на одной ЦА?                 │

│                                                                 │
│  ☐ 3. ОБМЕН РЕСУРСОВ                                          │

│        ☐ Энергия (уровень вовлечённости?)                      │
│     ☐ Что получаем:                                            │
│        ☐ Система генерирует ресурсы?                           │

│                                                                 │
│  ☐ 4. ПЕРЕХОД ФАЗА 1 → ФАЗА 2                                │
│     ☐ Достигнут ли порог "системы"?                            │
│     ☐ Система работает без постоянного ручного управления?      │

### Формулы для расчёта

```
ФОРМУЛА ФОКУСА:
Focus Score = (Ресурсы на одну ЦА) / (Общие ресурсы) × 100%
Цель: Focus Score > 80%

ФОРМУЛА ПЕРЕХОДА ФАЗ:
Phase 2 Threshold = (Доход от системы) > (Минимальная зарплата × 2)
Если: Системный доход > 2 × МРОТ → Фаза 2 достигнута
```

## МАТЕРИАЛ 2: Как заработать в Excel (Микробизнес)

### Суть методики

Статистика: 7 млн субъектов малого/среднего бизнеса в РФ, из них 6.3 млн — микробизнес (< 100 млн выручки).

### Ключевые принципы

1. **Мониторинг выручки** — знать свои цифры
2. **Челлендж-подход** — месячные циклы с заданиями
3. **Переход категории** — микро → малый (100 млн выручки)
4. **Конкретные метрики** — годовая выручка, обороты

### Чек-лист для применения методики

```
┌─────────────────────────────────────────────────────────────────┐
│  МИКРОБИЗНЕС — ЧЕК-ЛИСТ                                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ☐ 1. МЕТРИКИ КОМПАНИИ                                        │
│     ☐ Текущая годовая выручка: ___________ ₽                    │
│     ☐ Прогноз на конец года: ___________ ₽                     │
│     ☐ Средняя выручка в месяц: ___________ ₽                    │
│     ☐ Целевая выручка для след. фазы: ___________ ₽           │
│                                                                 │
│  ☐ 2. КАТЕГОРИЯ БИЗНЕСА                                       │
│     ☐ Микробизнес (< 100 млн)                                  │
│     ☐ Малый бизнес (100 млн — 1 млрд)                          │
│     ☐ Средний бизнес (1 млрд — 2 млрд)                          │
│                                                                 │
│  ☐ 3. ПЛАН ПЕРЕХОДА                                            │
│     ☐ Определён gap до следующей фазы?                          │
│     ☐ Есть месячные milestones?                                │
│     ☐ Выполняется ли план?                                      │
│                                                                 │
│  ☐ 4. ЧЕЛЛЕНДЖ-ПОДХОД                                         │
│     ☐ Поставлена цель на месяц?                                 │
│     ☐ Есть конкретное домашнее задание?                         │
│     ☐ Проводится ретроспектива?                                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Формулы для расчёта

```
ВЫРУЧКА:
Monthly Revenue = Annual Revenue / 12
Growth Rate = (Current Month - Previous Month) / Previous Month × 100%

GAP ДО СЛЕДУЮЩЕЙ ФАЗЫ:
Gap = Target Revenue (100M) - Current Revenue
Months to Goal = Gap / Monthly Growth (if linear)

МИНИМАЛЬНАЯ ЦЕЛЬ (по статистике):
Чтобы выйти из микробизнеса:
Target Monthly Revenue = 100,000,000 / 12 = ~8,333,333 ₽/месяц
```

## ИТОГОВАЯ ОЦЕНКА

### Техника "5 единичек" — Результат

| Критерий | Статус | Комментарий |
|----------|--------|-------------|
| Одна ЦА | ✅ Готово | Дети 10-16 лет |
| Фокус | ⚠️ Риск | Много фич, нужен сужение |
| Обмен ресурсов | ✅ Работает | Ценность > цена |
| Переход фаз | ⚠️ Не проверено | Нужен MVP |

**Вердикт по методике:** Идея **соответствует** принципам "5 единичек"

|----------|--------|-------------|
| Метрики определены | ✅ Готово | DAU, Revenue, NPS |
| Категория ясна | ✅ Готово | Микро → Малый |

| Челлендж-подход | ✅ Готово | Демо-урок |

```
