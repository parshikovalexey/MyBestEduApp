## 5 Единичек — Анализ

### 1. Одна ЦА?

```
СЛОЖНОСТЬ: ⚠️ Две РАЗНЫЕ ЦА

Дети и Предприниматели = РАЗНЫЕ потребности, РАЗНЫЕ мотивации, 
РАЗНЫЙ чек

Применяем технику "5 единичек":

БЫЛО: Одна ЦА (дети 10-16) ✅
СТАЛО: Две ЦА ❌

РЕШЕНИЕ: ВИЛИАРДНАЯ СТРУКТУРА
         ┌─────────────────────────────────────────────┐
         │           ОСНОВНОЙ ПРОДУКТ                  │
         │   (Общая платформа, общие процессы)         │
         │                    │                       │
         │   ┌────────────────┼────────────────┐    │
         │   │                │                │    │
         │   ▼                ▼                ▼    │
         │ СЕГМЕНТ 1      СЕГМЕНТ 2      СЕГМЕНТ 3  │
         │ (Дети)        (Бизнес)        (Профи)    │
         │ 199₽          1999₽          4999₽     │
         │                │                │        │
         │                ▼                ▼        │
         │   РАЗНЫЕ продукты, РАЗНЫЙ маркетинг       │
         └─────────────────────────────────────────────┘
```

**ВЫВОД:** Это НЕ противоречит "5 единичек" — общая платформа, 
но РАЗНЫЕ продукты для РАЗНЫХ сегментов. Фокус = СОЗДАТЕЛИ (все сегменты).

| Статья | Сумма |
|--------|-------|
| AI (дополнительно) | ~50₽/мес |
| Поддержка | ~30₽/мес |
| **ИТОГО VARIABLE** | **~80₽/подписчик** |

### Break-even

```
BREAK-EVEN POINT:
Fixed Costs / (Средний чек - Variable Cost)
20,000 / (500 - 80) = ~48 подписчиков

При среднем чеке 500₽ = ~50 подписчиков
При среднем чеке 300₽ = ~90 подписчиков
```

## Входная точка: Мини-курс по играм

### Как работает воронка

```
ДЛЯ ДЕТЕЙ:
                                    
[Демо-урок] → [Бесплатная игра] → [Сообщество FREE] 
                  ↓                                    ↓
            [Показывает друзьям]          [Хочу больше!] 
                  ↓                                    ↓
            [Родители видят РЕЗУЛЬТАТ]     [Подписка 299₽]
                  ↓                          
         [Родители: "Сколько?!"]          
                  ↓                          
            [Конверсия]

КЛЮЧЕВОЕ:
Демо-урок = "Зажигалочка" 
Результат за 5 минут → Показывает → Хотят ещё
```

### Конверсия

| Этап | Конверсия | Комментарий |
|------|----------|-------------|
| Демо → Бесплатный доступ | 30% | Зарегистрировались |
| Бесплатный → Подписка | 5-10% | Платят |
| Подписка → Реферал | 20% | Приглашают друзей |

**CAC (Customer Acquisition Cost):**
- Органический: ~50₽ (рефералы)
- Реклама: ~200-500₽ (нужно тестировать)

**LTV (Lifetime Value):**
- Дети: 299₽ × 6 месяцев = ~1,800₽
- Бизнес: 1999₽ × 12 месяцев = ~24,000₽
