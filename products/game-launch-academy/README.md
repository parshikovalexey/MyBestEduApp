## Описание продукта

Образовательная платформа для обучения детей (10-16 лет) созданию игр для ВКонтакте с использованием AI. Включает курс, AI-тьютора, VK Mini Apps и партнёрскую программу.

**Ключевые элементы:**
- Мини-курс по созданию игр
- Демо-игры для ВК с рекламой
- AI-тьютор (куратор)
- VK Mini App платформа
- Сообщество ВК
- Партнёрская программа с кэшбеком

## Демо-урок: Скачиваем Opencode

**Цель:** Показать "вау-эффект" за 5 минут с минимумом действий.

**Сценарий:**
1. Скачиваем opencode
2. Устанавливаем
3. Пишем интересный промпт
4. Получаем ответ
5. Понимаем: "Это может сделать... но как именно?"

**Ключевой инсайт:**
- Главное = "замануха"
- Простое активное действие → быстрый результат
- Но всё равно непонятно как делать самому

**Результат:** Хочу научиться → покупаю курс
