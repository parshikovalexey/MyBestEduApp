## Executive Summary

**AI Creator Hub** — это мультисегментное онлайн-сообщество для людей, которые хотят создавать реальные продукты с помощью ИИ.

**Фокус:** Объединяем людей, которые хотят **СОЗДАВАТЬ** (а не только потреблять), через ИИ как инструмент.

**Модель:** Подписочная (299₽ — 4999₽/мес)

## Сегменты Сообщества

### Сегмент 1: Дети (10-16 лет)

**Направление:** Создание игр в ВКонтакте

| Параметр | Значение |
|----------|---------|
| Продукт | Игры для ВК |
| Цена | 199-399₽/мес |
| Вход | Демо-урок (бесплатно) |
| Результат | Работающая игра в ВК |
| Мотивация | Хвастовство друзьям |
| Плательщик | Родители |

**User Flow:**
1. Видит игру в ВК
2. Хочет свою
3. Делает демо-урок (5 мин)
4. Получает игру!
5. Показывает друзьям
6. Хочет ещё → Подписка
7. Ребёнок допекает родителей

---

### Сегмент 2: Предприниматели

**Направление:** Создание приложений и сервисов

| Параметр | Значение |
|----------|---------|
| Продукт | Приложения, автоворонки, сервисы |
| Цена | 1999-4999₽/мес |
| Вход | Мини-курс |
| Результат | Готовый продукт для бизнеса |
| Мотивация | Прибыль, автоматизация |
| Плательщик | Сам |

**User Flow:**
1. Хочет автоматизировать/создать продукт
2. Проходит мини-курс
3. Создаёт первый продукт
4. Видит результат
5. Оформляет подписку для доступа к материалам
6. Развивает бизнес

## Расходы

### Ежемесячные (Fixed)

| Статья | Сумма |
|--------|-------|
| Хостинг (сервер) | 5,000₽ |
| Домены / SSL | 1,000₽ |
| Платформа сообщества | 3,000₽ |
| AI (лимиты) | 10,000₽ |
| Инструменты | 2,000₽ |
| **ИТОГО** | **21,000₽/мес** |

### Переменные (на подписчика)

| Статья | Сумма |
|--------|-------|
| AI (доп.) | 50₽ |
| Поддержка | 20₽ |
| **ИТОГО** | **70₽/подписчик** |

### Break-even

```
Fixed Costs / (Средний чек - Переменные)
21,000 / (500 - 70) = ~49 подписчиков

При 50+ подписчиках = выходим в прибыль
```
