## Хуки для разных ЦА

### 1. Для Родителей (основная боль)

```
"Ксюша, 12 лет, сделала эту игру за 20 минут.
Твой ребенок может также."

Переход: Пост в ВК → Лендинг → [Купить курс 2,990₽]
```

**Боль родителя:** "Я не понимаю в играх, но хочу чтобы ребенок был занял чем-то полезным"

---

### 2. Для Детей (мотивация заработка)

```
"Ксюша заработала 10,000₽ за неделю, создав эту игру.
Ты можешь так же."

Переход: Пост в ВК → Лендинг → [Купить курс]
```

**Боль ребенка:** "Хочу крутой телефон/скины, но нет денег"

---

### 3. Для Бабушек/Дедушек (подарок)

```
"Подари внуку не просто игрушку, а умение СОЗДАВАТЬ игры.
Это развивает логику и творчество."

Переход: Пост → Лендинг → [Подарить курс]
```

**Боль бабушки:** "Хочу быть полезной, дать что-то ценное"

## Потенциал проекта

### Почему может сработать

```
✓ Проверенная модель: Coursename, Skillbox, Udemy работают
✓ Ниша: games for VK = мало конкурентов
✓ Результат: Игра за 20 минут = "вау-эффект"
✓ Цена: 2,990₽ = не дорого для родителей
✓ Лендинг: Простой, понятный, один оффер
✓ AI: Делает курс создаваемым для детей
```

### Почему может НЕ сработать

```
✗ Нет трафика: Не можем получить просмотры
✗ Конкуренция: Кто-то уже делает похожее
✗ Качество: AI генерирует плохой контент
✗ Поддержка: Не справляемся с вопросами
✗ Возврат: Блогер курс не работает
```

## Вердикт

**Стратегия:**
1. Запустить курс (минимальный, 5 уроков)
2. Дать рекламу на лендинг
3. Если продажи идут → расширять
4. Если есть спрос → добавить сообщество

Создать продающий лендинг и запустить тестовую рекламу на 5,000₽
