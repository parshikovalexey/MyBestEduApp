# MVP Specification: Game Launch Academy

**Дата:** 17.04.2026  
**Версия:** 2.0 MVP (Подписочная модель)  
**Status:** READY FOR DEVELOPMENT

---

## Overview

**ОБНОВЛЁННАЯ МОДЕЛЬ:**

- Подписочная модель (299-599₽/мес)
- Сообщество + Платформа (VK Mini App)
- Маркетплейс игр (внутри платформы)
- Валидация игр для публикации в ВК
- Партнёрская программа

**Цель MVP:** Проверить интерес аудитории и получить первых платящих подписчиков.

## Architecture Decisions

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   ПЛАТФОРМА (VK MINI APP):                                │
│   ├── Контент уроков — статика                            │
│   ├── Прогресс — VK Storage                               │
│   ├── Игры — генерируются как Mini Apps                   │
│   └── Галерея — VK Community посты                        │
│                                                             │
│   МИНИ-БЭК (необходим):                                  │
│   ├── AI-тьютор — нужен сервер для OpenAI API            │
│   ├── Оплаты — webhook от VK Donut / ЮKassa             │
│   ├── Генерация Mini App из игры                          │
│   ├── Валидация игр для публикации                        │
│   └── Партнёрская программа (отслеживание дохода)         │
│                                                             │
│   STORAGE STRATEGY:                                        │
│   ├── VK Storage API → прогресс, настройки               │
│   ├── PostgreSQL → пользователи, игры, подписки           │
│   └── Mini Apps → хостинг игр (VK инфраструктура)        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**Ключевое решение:** Игры = Mini Apps. Хостинг бесплатный (VK). Публикация в каталог ВК = охват 80M+.
