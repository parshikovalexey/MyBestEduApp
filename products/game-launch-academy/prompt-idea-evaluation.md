## ЧАСТЬ 1: АНАЛИЗ ИДЕИ

### Описание проекта

- **Название:** Game Launch Academy
- **Суть:** Курс для детей (10-16 лет) по созданию игр в ВКонтакте с помощью AI
- **Цена:** 2,990₽
- **Бизнес-модель:** Простой платный курс (без подписки, без сообщества)
- **Воронка:** Игра-замануха → Лендинг → Покупка → Курс

### Вопросы для анализа:

1. **Описание (1-10):** Насколько понятно описан продукт? Родитель поймёт за 10 секунд?

2. **Боль (1-10):** Какую боль решает? Родителю важно чтобы ребёнок был занят?

3. **Решение (1-10):** Курс — это решение? Или игрушка?

4. **Уникальность (1-10):** Почему именно этот курс, а не другой?

5. **Цена (1-10):** 2,990₽ — дорого или дёшево? Почему?
