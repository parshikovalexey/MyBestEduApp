## Ключевой подход к AI тьютору

### Главный принцип
> **AI формирует НАВЫК РЕШЕНИЯ ПРОБЛЕМ, а не просто объясняет материал**

### Как это работает

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   РЕБЕНОК СТАЛКИВАЕТСЯ С ПРОБЛЕМОЙ                       │
│   (не понял материал / не может решить задачу)             │
│                                                             │
│                         ▼                                  │
│   ┌───────────────────────────────────────────────────┐    │
│   │                                                   │    │
│   │   AI НЕ ПРОСТО ОБЪЯСНЯЕТ                        │    │
│   │                                                   │    │
│   │   AI УЧИТ РЕШАТЬ ПРОБЛЕМЫ:                      │    │
│   │                                                   │    │
│   │   1. Локализовать проблему                       │    │
│   │      "Где именно застрял? Что не понятно?"       │    │
│   │                                                   │    │
│   │   2. Найти решение                               │    │
│   │      "Давай попробуем по-другому..."            │    │
│   │                                                   │    │
│   │   3. Проверить что работает                      │    │
│   │      "Реши похожую задачу сам"                  │    │
│   │                                                   │    │
│   │   4. Закрепить навык                            │    │
│   │      "Теперь ты умеешь решать такие задачи"     │    │
│   │                                                   │    │
│   └───────────────────────────────────────────────────┘    │
│                                                             │
│                         ▼                                  │
│   РЕБЕНОК ПОЛУЧАЕТ НАВЫК,                                  │
│   который применит в будущем                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### Это про мета-навыки

Ребенок учится:
- **Не сдаваться** перед трудностью
- **Анализировать** где именно проблема
- **Искать** разные способы решения
- **Проверять** свои решения
- **Учиться** из ошибок

Это применимо НЕ только к математике, а к ЛЮБОЙ проблеме!

## Форматы взаимодействия

### 1. Текстовый чат
- Ребенок пишет вопрос или описание проблемы
- AI ведет диалог, задает уточняющие вопросы
- Помогает разобраться шаг за шагом

### 2. Голосовой ассистент
- Разговор с AI (как Siri, но умный)
- Особенно для младших детей
- Можно делать упражнения на ходу

### 3. Интерактивные задания
- AI адаптирует сложность
- Мгновенная обратная связь
- Объяснение ошибок

### 4. Исследовательские проекты
- AI помогает исследовать темы
- Задает вопросы для думания
- Помогает структурировать мысли

## MVP: AI Тьютор для ребенка

### Основные функции MVP

1. **Диалоговый интерфейс**
   - Текстовый и голосовой ввод
   - AI ведет беседу, а не выдает ответы

2. **Локализация проблемы**
   - AI задает вопросы чтобы понять где затык
   - Выявляет КОРЕНЬ проблемы, а не симптомы

3. **Мульти-объяснения**
   - Разные способы объяснить одно и то же
   - Адаптация под стиль ученика

4. **Практика с закреплением**
   - Не просто объяснить, а убедиться что понял
   - Дать похожие задания для закрепления

5. **Progress tracking**
   - Отслеживание развития навыков
   - Видимость прогресса

### Что НЕ входит в MVP

- Dashboard для родителей
- Инструменты для педагога
- Полная интеграция с школьной программой
- Синхронизация с учителями

### Метрики успеха MVP

- Retention rate (% детей возвращается)
- Engagement time (сколько взаимодействуют)
- Skill improvement (растут ли навыки)
- NPS ( Net Promoter Score от детей)
