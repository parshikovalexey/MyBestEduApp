## Цель дерева будущего

Найти решения для корневой проблемы:
> **"Образование фокусируется на передаче знаний, а не на развитии навыков"**

---

## Целевое состояние

### Главная цель:
> **Создать образовательную экосистему, которая:**
> 1. Развивает навыки информационного века
> 2. Персонализирует обучение для каждого
> 3. Дает мгновенную поддержку при трудностях
> 4. Соединяет всех участников (педагог-ученик-родитель)
> 5. Готовит к постоянному обучению, а не к профессии

## Как это меняет роли участников

### Педагог

| БЫЛО | СТАНЕТ |
|------|--------|
| Транслятор программы | Фасилитатор развития |
| Оценивает через оценки | Видит реальный прогресс |
| Один на 30+ учеников | AI помогает с персонализацией |
| Заполняет отчеты | Получает insights автоматически |
| "Мы дали" | "Мы помогаем каждому" |

### Ученик

| БЫЛО | СТАНЕТ |
|------|--------|
| Пассивный потребитель | Активный исследователь |
| Учит для оценки | Учит для понимания |
| Боится ошибок | Может ошибаться без осуждения |
| Один с вопросом | Получает помощь мгновенно |
| Не понимает зачем | Видит применение знаний |

### Родитель

| БЫЛО | СТАНЕТ |
|------|--------|
| Не понимает что происходит | Видит прогресс ребенка |
| Не знает как помочь | Получает рекомендации |
| Конфликты из-за учебы | Партнер в образовании |
| Перекладывает на школу | Вовлечен в процесс |
| Платит за репетиторов | Использует AI поддержку |

## Ключевые изменения

### 1. Фокус: От знаний к навыкам
```
ЗНАЕТ → УМЕЕТ → ПОНИМАЕТ ЗАЧЕМ
(запоминание) (применение) (осмысленность)
```

### 2. Оценка: От оценок к progress
```
ОЦЕНКА → ПОНИМАНИЕ → РОСТ → НАВЫКИ
(статично)  (динамично)
```

### 3. Поддержка: От изоляции к connected
```
УЧЕНИК ОДИН → AI ПОДДЕРЖКА → СВЯЗЬ С УЧИТЕЛЕМ → ВОВЛЕЧЕННОСТЬ РОДИТЕЛЕЙ
```

### 4. Роль учителя: От транслятора к наставнику
```
ПРОГРАММА → УЧЕНИК    →    УЧЕНИК ← УЧИТЕЛЬ ← AI
(слепое следование)     (поддержка и направление)
```

## Гипотезы для проверки

### Гипотеза 1: Навыки важнее знаний
> Мы полагаем, что фокус на развитии навыков (критическое мышление, самообучение) даст детям лучшую подготовку к информационному веку, чем традиционный фокус на знаниях.

**Как проверить:** Исследовать результаты программ oriented на навыки (Montessori, inquiry-based learning)

### Гипотеза 2: AI tutor как решение проблемы пробелов
> Мы полагаем, что доступный 24/7 AI тьютор решит проблему "если не понял - не освоишь", потому что помощь будет доступна мгновенно.

**Как проверить:** A/B тестирование с контрольной группой

### Гипотеза 3: Вовлеченность родителей
> Мы полагаем, что прозрачность прогресса (dashboard для родителей) увеличит вовлеченность родителей и улучшит результаты детей.

**Как проверить:** Пилот с родителями, измерение engagement

### Гипотеза 4: Роль педагога как фасилитатора
> Мы полагаем, что педагоги будут более удовлетворены работой если смогут быть наставниками, а не трансляторами программы.

**Как проверить:** Интервью с педагогами, пилотные программы
