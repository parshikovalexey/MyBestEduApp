# Копилка ниш для бизнеса

## Зафиксированные ниши

### 1. IT-аутсорсинг с обучением штата (IT + образование)
- **Суть:** Обычный IT-аутсорсинг, но с УТП = "обучаем сотрудников клиента работать с IT-системами, чтобы решать 70% вопросов сами"
- **Почему подходит:** IT + образование = идеально под твой опыт
- **Масштаб:** От города до страны
- **Статус:** Фиксировано

### 2. B2B-платформа для обучения сотрудников (SaaS + образовательные методики)
- **Суть:** LMS для малого бизнеса — "внедрим за неделю, сотрудники обучаются сами"
- **Почему подходит:** Качественное обучение = снижение текучки = УТП
- **Масштаб:** От малого бизнеса до корпораций
- **Статус:** Фиксировано

### 3. Консалтинг по оптимизации бизнес-процессов + обучение сотрудников
- **Суть:** "Оптимизируем + обучаем команду внедрять без нас"
- **Почему подходит:** IT + аналитика + образование — идеальное сочетание
- **Масштаб:** От города до страны
- **Статус:** Фиксировано

### 5. Агентство по разработке корпоративных курсов "под ключ"
- **Суть:** "Разрабатываем курсы для корпоративного обучения"
- **Почему подходит:** Образование = ядро, твой опыт в разработке материалов
- **Масштаб:** От города до страны
- **Статус:** Фиксировано

### 6. Агентство по внедрению ИИ в бизнес-процессы + обучение
- **Суть:** "Внедряем ИИ + учим сотрудников работать с ним"
- **Почему подходит:** Горячая тема, IT-бэкграунд + образовательные навыки
- **Масштаб:** От города до страны
- **Статус:** Фиксировано

## Дополнительные варианты (предложены, не зафиксированы)

### 4. Сервис по подбору и адаптации персонала (HR-tech)
- **Суть:** Найм + адаптация 90 дней, обучаем кандидата до выхода
- **Рынок:** HR-tech ~80млрд

### 5. Маркетинговое агентство с образовательным продуктом
- **Суть:** "Мы не просто делаем — мы учим ваш маркетинг работать без нас"

### 6. Сервис автоматизации для малого бизнеса
- **Суть:** Внедрение + обучение + видео-уроки

### 7. Агентство по внедрению CRM/ERP с "школой клиента"
- **Суть:** "Клиент получает обученных пользователей, а не просто ПО"

### 8. Сервис бухгалтерии/юриспруденции с обучением
- **Суть:** "Мы не просто ведем — мы учим вас понимать финансы"
